## 🔐 Рекомендации по использованию

### 1. Переменные окружения

```bash
# Токен HuggingFace (не хранить в коде!)
export HF_TOKEN=your_token_here

# Имя окружения
export RKLLM_ENV_NAME=rkllm
```

### 2. Права доступа

```bash
# Ограничить доступ к рабочим файлам
chmod 700 /path/to/rkllm-tools-fork/workdir
```

### 3. Сетевая безопасность

```bash
# Проверка SSL сертификатов при загрузке
python download_hf_model.py download <model> --verify-ssl
```
